<h2><a href="https://www.geeksforgeeks.org/problems/sort-by-absolute-difference-1587115621/1?page=1&category=Arrays&difficulty=Easy&status=unsolved&sortBy=submissions">Sort by Absolute Difference</a></h2><h3>Difficulty Level : Difficulty: Easy</h3><hr><div class="problems_problem_content__Xm_eO"><p><span style="font-size: 18px;">You are given a number<strong> k</strong> and array <strong>arr.</strong> Your task is to rearrange the elements of array <strong>arr</strong> according to the absolute difference with <strong>k</strong>, i.e., an element having minimum difference comes first, and so on.<br><strong>Note:</strong> If two or more elements are at equal distances arrange them in the same sequence as in the given array.</span></p>
<p><span style="font-size: 18px;"><strong>Examples:</strong></span></p>
<pre><span style="font-size: 18px;"><strong>Input</strong>: k = 7, arr[] = [10, 5, 3, 9, 2]
<strong>Output</strong>: [5, 9, 10, 3, 2]
<strong>Explanation</strong>: Sorting the numbers according to the absolute difference with 7, we have array elements as 5, 9, 10, 3, 2.</span></pre>
<pre><span style="font-size: 18px;"><strong>Input</strong>: k = 6, arr[] = [1, 2, 3, 4, 5]
<strong>Output</strong>: [5, 4, 3, 2, 1]<br><strong>Explanation</strong>: Sorting the numbers according to the absolute difference with 6, we have array elements as 5, 4, 3, 2, 1.<br></span></pre>
